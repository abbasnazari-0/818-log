# 818 Stylist - فروشگاه آنلاین

## تغییرات جدید

### 1. تغییر نام برند
- نام پروژه از "GlobalLink Logistics" به **818 Stylist** تغییر کرد
- زیرعنوان: **فروشگاه آنلاین**

### 2. صفحه ردیابی عمومی
یک صفحه زیبا و کاربرپسند برای ردیابی مرسولات بدون نیاز به لاگین ساخته شد.

**آدرس:** `/tracking` یا `http://localhost:3001/tracking`

**ویژگی‌ها:**
- ✅ بدون نیاز به لاگین
- ✅ جستجو با کد رهگیری (مثال: `PKG-10001`)
- ✅ نمایش اطلاعات کامل بسته
- ✅ نوار پیشرفت ارسال
- ✅ تایم‌لاین مراحل ارسال
- ✅ طراحی زیبا با گرادیانت‌های رنگی
- ✅ پشتیبانی کامل از فارسی

**نحوه استفاده:**
1. به آدرس `/tracking` بروید
2. کد رهگیری را وارد کنید (مثلاً `PKG-10001`)
3. روی دکمه "رهگیری" کلیک کنید
4. اطلاعات کامل مرسوله نمایش داده می‌شود

### 3. احراز هویت با پسورد
سیستم لاگین حالا نیاز به ایمیل **و** پسورد دارد.

**اطلاعات لاگین:**
- Admin: `admin@globallink.com` / `admin123`
- China Agent: `agent.china@globallink.com` / `china123`
- UAE Agent: `agent.uae@globallink.com` / `uae123`
- Iran Agent: `agent.iran@globallink.com` / `iran123`
- Customer: `customer@example.com` / `customer123`

### 4. قابلیت حذف سفارش
- دکمه حذف در لیست سفارشات اضافه شد
- فقط Admin و China Agent می‌توانند سفارش حذف کنند
- تایید دوباره قبل از حذف

## دستورات مفید

```bash
# اجرای پروژه
npm run dev

# ساخت نسخه تولید
npm run build

# بارگذاری داده‌های اولیه
npm run seed
```
